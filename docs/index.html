纯粹，但不简陋

从 ANC、低延迟、音质偏好到手势和日志导出——只做耳机控制，但把细节做稳。

🔌

离线直连

通过蓝牙 SPP 与耳机通信，无需华为账号、云服务或网络权限。

🎛️

ANC 核心控制

支持关闭、降噪、透传和子模式映射，修复数字子选项与状态回跳。

⚡

低延迟 / 音质偏好

连接后优先读取核心状态，慢项后台同步，减少等待和误判。

✋

手势设置

双击、三击、长按、滑动等手势逐步适配，支持日志反馈校准。

🔮

液态玻璃 UI

Haze 2.0 主线视觉、暗色可读性保护、边缘光学和自定义壁纸。

🧾

日志友好

内置调试日志与分享入口，方便测试者反馈不同型号和固件表现。

支持状态

能力表持续校准中。不同固件和 Android 蓝牙栈会影响实际表现，日志非常有价值。

FreeBuds 6i已实测主力开发设备，核心状态 fast init，ANC / 低延迟 / 音质持续调优。
FreeBuds 7i测试修复中v3.6.0 起优化 ANC 初始化、透传子模式与状态回跳。
FreeBuds 5i / 4i待更多日志已配置能力表，欢迎提供 ANC、手势、电量和音质偏好日志。
Pro / SE / FreeClip / FreeLace重点招募型号差异大，需测试者帮助确认协议行为。

测试反馈建议

提交 Issue 时请附上耳机型号、固件、手机系统、App 版本和“分享日志”导出的日志文件。

如果看到某个功能“先没有、后出现”，通常说明该 handler 正在后台补齐；新版设备页会显示「后台同步中」。
