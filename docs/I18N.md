# fxxkHilife i18n 接口说明

当前版本只内置默认中文文案，不提供其它语言文件。

## 入口

代码入口位于：

```text
app/src/main/java/com/freebuds/controller/i18n/I18n.kt
```

核心接口：

```kotlin
interface I18nProvider {
    val localeTag: String
    fun t(key: String, vararg args: Any?): String
}
```

普通 Kotlin/Service/Tile 场景：

```kotlin
I18n.t("settings.title")
I18n.t("settings.saved_devices", count)
```

Compose 场景：

```kotlin
Text(i18n("settings.title"))
```

## 后续翻译建议

1. 先在 `I18n.kt` 中补充 key，不要直接散落硬编码字符串。
2. 翻译者可新增语言 provider，例如 `EnglishI18nProvider`。
3. 如需运行时切换语言，可在 App 根部使用 `CompositionLocalProvider(LocalI18n provides provider)` 注入对应 provider。
4. 如果后续决定改回 Android resource 方案，也可以保留 key 命名，并把 `I18nProvider` 实现转接到 `Context.getString()`。

## key 命名规则

建议使用分层命名：

```text
common.*
settings.*
device.*
notification.*
tile.*
gesture.*
scan.*
permission.*
```

示例：

```text
settings.auto_low_latency
tile.connection_failed_retry
notification.channel.bluetooth_status
```
